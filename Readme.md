Titre: Reconnaissance faciale

Description: 
Lors du démarrage de la page "index.html", l'utilisateur trouvera le bouton "AnalyseFace qui ouvrir sa caméra vidéo(accessible par un micro-ordinateur, téléphone intelligent, ...)aprés avoir autoriser l'utilisation de sa caméra
et prendre une photo qui s'affichera sur l'écran ensuite l'utilisateur peut avoir une description de son visage(age, couleur des cheveux, couleur des yeux, émotions...) selon sa photo déja prise

Les outils utilisés :
    - Serveur Web : hébergment local ou hébergement web en ligne
    - Editeur: Microsoft visuel studio code
    - Languages: HTML5, Javascript, Css
    - FrameWork: Jquery
    - API: api.cognitive.microsoft.com
